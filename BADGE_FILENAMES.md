## Tips
- Badges don't need to be exact size вЂ” they're rescaled to fit (20px in table, 20px in fixture panel, 20px in match HUD)
- Square PNGs work best (e.g. 64Г—64, 128Г—128, 512Г—512)
- Transparent background recommended
- If a file is missing, the emoji badge shows instead вЂ” no crash, no broken layout
- You can add badges one club at a time; others will keep using emoji until you add theirs
